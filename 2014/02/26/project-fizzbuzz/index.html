Hoggetowne Hack & Project - FizzBuzz

Hello! Its been sometime since I wrote my blog. It has been a hectic week. I went to Hoggetowne hackathon came in second which is awesome. I got to meet some pretty good developers there.

Anyways Today I found something interesting... ever heard of FizzBuzz its ridiculous. Its kind of a simple hack to find weak programmers. The text goes like this :

Write a program that prints the numbers from 1 to 100. But for multiples of three print "Fizz" instead of the number and for the multiples of five print "Buzz". For numbers which are multiples of both three and five print "FizzBuzz".

Thats weird anyone can do it...right? nope, some say its like a pre-screen for anyone who is a weak programmer. Check this blog. I didn't want to be one of "those guys" :P

So I wrote the code and It made feel like a programmer again "one step above a sucky one atleast I think"....

Anyways heres the code :

public static void main(String args[]) {
    for(int i=1; i< 101; i++) {
        if(i%3==0 && i%5==0) {
            System.out.print("FizzBuzz  ");
        } else if(i%3==0) {
            System.out.print("Fizz  ");
        } else if(i%5==0) {
            System.out.print("Buzz  ");
        }else {
            System.out.print(i+"    ");
        }       
    }
}

Now I know you can improve the if conditions and make it more optimized....But ahhhh! I feel lazy!...

Its fun writing about silly but important stuff though!......
