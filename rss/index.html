boolean solveSudoku(int[][] sudokumatrix) { 
    boolean solved = false;
    for(int i = 0; i< 9; i++) {
        for(int j =0; j <9; j++) {
            if(sudokumatrix[i][j] == 0) {
                int k = 1;
                count++;
                for( k = 1; k < 10; k++) {
                    boolean alreadyPresent = false;

                    // check in the row.
                    for(int l= 0 ; l< 9; l++) {
                        if(sudokumatrix[i][l] == k && l != j){
                            alreadyPresent = true;
                            break;
                        }
                    }

                    // check in the column.
                    if(!alreadyPresent){
                        for(int l= 0; l < 9; l++) {
                            if(sudokumatrix[l][j] == k && l != i) {
                                alreadyPresent = true;
                                break;
                            }
                        }
                    }

                    // check in the box.
                    if(!alreadyPresent){
                        int m = 0, n= 0;
                        if(0 <= i && i < 3 ) {
                            m = 0;
                        }else if(3<=i && i < 6){
                            m = 3;
                        }else if(6<=i && i < 9){
                            m = 6;
                        }

                        if(0 <= j && j < 3 ) {
                            n = 0;
                        }else if(3<=j && j < 6){
                            n = 3;
                        }else if(6<=j && j < 9){
                            n = 6;
                        }

                        for(int m1 = m; m1 < m+3; m1++){
                            for(int n1 = n; n1<n+3; n1++){
                                if(m1!= i && n1 !=j && sudokumatrix[m1][n1]== k){
                                    alreadyPresent = true;
                                    break;
                                }
                            }
                            if(alreadyPresent)
                                break;
                        }
                    }
                    if(!alreadyPresent){
                        sudokumatrix[i][j] = k;
                        if(solveSudoku(sudokumatrix)) {
                        // k value solved the sudoku for that cell.
                            return true;
                        }else {
                            // reverting since this k value doesnt solve the sudoku.
                            sudokumatrix[i][j] = 0;
                            solved = false;
                        }
                    }
                }
                // If you reach this part it means there is no value that we can put in this cell.
                if (k == 10){
                    return false;
                }
            }   
        }
    }
    // If you reach this part that means there is no empty cell in the sudoku is.
    return true;
}

Pagerank is like finding the most popular web page among its peers and present it to the user as the most relevant page. This popularity is calculated by the hyper links. The formula for calculating page rank is

PR(A) = (1-d)/N + d( sum of all( PR(I)/NO(I)))

where
A - It is the page for that we want to calculate page rank for.

d - 0.85 dampening factor. ( There is lot of research that went into calculating this value which cannot be covered here. So do your own research on this. The only thing I can mention here is this value causes the pagerank values for any A to converge as the number of iterations increases)

N - Total number of pages for which you want to calculate pagerank.

I - is the inlink page for A i.e you can reach A with a hyper link from I.

NO(I) - number of outlinks from I i.e. number of hyper links present on I.

For more information I would suggest you to look it up on wiki.

So for my project I took the wiki dumps and calculated the page rank.

The code can be found at github

Trivia : We think page in pagerank refers to web pages but its not. It's Page as in Larry Page, CEO of Google.
